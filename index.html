Home Run Derby

About This Project

Hello! Welcome to this fun and simple game.

This game is a simple home run derby. Choose between two stadiums and five difficulties. Currently, it only has an offline mode, but an online version is in creation, and may work soon.

Physics and 3d graphics on the web are very difficult to run well and performantly, so issues may occur, or it may be slow. I've had the best performace on chrome.

Thank you to mlb for the stadium files, as well as Antti Luode & Punch Deck for the incredible CC music. Additionally, I used Three.js for rendering, Ammo.js for physics, and GSAP for some animations.

Thank you for playing. Check out this and more projects on Github.
